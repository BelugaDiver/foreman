## Constitution Check

*GATE: Must pass before Phase 0 research. Re-check after Phase 1 design.*
*Reference: `.specify/memory/constitution.md`*

- [ ] **I. Layered Architecture** — every new entity follows migrations → models → schemas → repositories → endpoints → main.py; worker reuses foreman layers.
- [ ] **II. Raw SQL** — no ORM/query-builder imports; `sql()` helper used everywhere; `ALLOWED_UPDATE_FIELDS` guard for partial UPDATEs.
- [ ] **III. Async-First** — no blocking I/O in async functions without `asyncio.to_thread()`.
- [ ] **IV. Event-Driven** — generation work dispatched via `QueueProtocol`, not called synchronously from the API.
- [ ] **V. Protocols** — new external integrations implement and are accessed through an ABC protocol class.
- [ ] **VI. Test Layers** — unit tests use in-memory mocks + `monkeypatch`; integration tests use testcontainers; 85% coverage gate.
- [ ] **VII. Observability** — OTEL spans added for significant operations; no PII or prompt content in span attributes; audit events logged.
- [ ] **VIII. Security** — all owned-resource queries include `AND user_id=$n`; no `str(exc)` in responses; no secrets in code; SSRF checks on remote URL fetches.
