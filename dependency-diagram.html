⚠️ Missing / Needed Components

🔄 Background Worker / Queue System
Currently, generations are created with status="pending" but there's no worker processing them. Need Celery, Redis, or similar to poll for pending generations and call external AI service.
🔗 Webhook Callbacks
No webhook system to notify users when generation status changes. Need table + endpoint to register webhooks and async worker to POST on completion.
🤖 AI Design Client
No client/integration for actual room design generation (e.g., Stable Diffusion, DALL-E, Midjourney, ControlNet). Need an abstraction layer to call external AI APIs and update generation.status/output_image_url.
📊 Analytics / Metrics Dashboard
No dedicated analytics endpoints. Consider adding /v1/analytics/* for usage stats.
🔔 Notifications Service
No email/push notification integration for generation completion or failures.
📝 API Rate Limiting
No rate limiting middleware. Consider adding slowapi or custom Redis-based rate limiting.
🔐 API Key Management
No API key system for programmatic access. Users currently managed only via gateway X-User-ID.
